Homomorfismos de grupos¶

Um homomorfismo de um grupo G em um grupo H é uma função \(f:G\longrightarrow H\) tal que \(f(x\cdot y)=f(x)\cdot f(y)\) para todos \(x,y\in G\).

Um homomorfismo de grupos \(f:G\longrightarrow H\) é chamado monomorfismo, epimorfismo ou isomorfismo conforme a função f seja, respectivamente, injetora, sobrejetora ou bijetora.

Nota

Se \(G, H\) são grupos, então se escreve \(G\cong H\) se \(G\) é isomorfo a \(H\).

Exemplos¶

  1. O logarítmo

    \[\log:(0,\infty)\longrightarrow\mathbb{R},\quad x\longmapsto\log{x},\]

    é um isomorfismo do grupo multicativo \((0,\infty)=\{x\in\mathbb{R}:x>0\}\) dos números reais positivos no grupo aditivo \(\mathbb{R}\) de todos os números reais. Além de bijetiva, a função \(\log\) satisfaz:

    \[\log{(x\cdot y)}=\log{x}+\log{y},\]

    para todos \(x,y\in(0,\infty)\);

  2. O determinante de matrizes

    \[\mathrm{GL}(n,\mathbb{R})\longrightarrow\mathbb{R}^\star,\quad A\longmapsto\det{A},\]

    é um epimorfismo do grupo multiplicativo \(\mathrm{GL}(n,\mathbb{R})=\{A\in\mathrm{Mat}(n\times n,\mathbb{R}):\det{A}\neq0\}\) das matrizes inversíveis no grupo multiplicativo \(\mathbb{R}^\star=\mathbb{R}\setminus\{0\}\) dos números reais diferentes de zero, pois \(\det:\mathrm{GL}(n,\mathbb{R})\longrightarrow\mathbb{R}^\star\) é sobrejetiva e

    \[\det{(A\cdot B)}=\det{A}\cdot\det{B},\]

    sempre que \(A,B\in\mathrm{GL}(n,\mathbb{R})\);

  3. O traço

    \[\mathrm{Mat}(n\times n,\mathbb{R})\longrightarrow\mathbb{R},\quad A\longmapsto\mathrm{tr}\,A=\sum_{i=1}^n\langle Ae_i,e_i\rangle,\]

    é um epimorfismo do grupo aditivo \(\mathrm{Mat}(n\times n,\mathbb{R})=\{(a_{ij})_{i,j=1}^n:a_{ij}\in\mathbb{R},\,\forall\,\,i,j\in \{1,\ldots,n\}\}\) das matrizes \(n\times n\) com coeficientes reais no grupo aditivo \(\mathbb{R}\) dos números reais, pois \(\mathrm{tr}\) é sobrejetiva e

    \[\mathrm{tr}\,(A+B)=\mathrm{tr}\,A+\mathrm{tr}\,B,\]

    sempre que \(A,B\in\mathrm(n\times n,\mathbb{R})\).

Propriedades¶

Proposição

Seja \(f:G\longrightarrow H\) um homomorfismo de grupos. Então:

  1. \(f(e_G)=e_H\);

  2. \(f(x^{-1})=f(x)^{-1}\) para todo \(x\in G\);

  3. \(f(x^n)=f(x)^n\) para todos \(x\in G\) e \(n\in\mathbb{N}\);

  4. \(f(x^y)=f(x)^{f(y)}\) para todos \(x,y\in G\).

Prova

  1. Seja \(y=f(e_G)\in H\). Observe que:

    \[y=f(e_G)=f(e_G\cdot e_G)=f(e_G)\cdot f(e_G)=y\cdot y=y^2,\]

    de onde segue que

    \[e_H=y\cdot y^{-1}=y^2\cdot y^{-1}=y;\]

  2. Se \(x\in G\) então

    \[e_H=f(e_G)=f(x\cdot x^{-1})=f(x)\cdot f(x^{-1}),\]

    de onde segue que \(f(x^{-1})=f(x)^{-1}\) pela unicidade do elemento inverso em H;

  3. Se \(x\in G\) então

    \[f(x^2)=f(x\cdot x)=f(x)\cdot f(x)=f(x)^2,\]

    e mais geralmente, se \(f(x^n)=f(x)^n\) para algum \(n\in\mathbb{N}\), \(n\geqslant2\), então

    \[f(x^{n+1})=f(x^n\cdot x)=f(x^n)\cdot f(x)=f(x)^n\cdot f(x)=f(x)^{n+1},\]

    de onde segue que \(f(x^n)=f(x)^n\) para todo \(n\in\mathbb{N}\) pelo Primeiro Princípio de Indução Matemática;

  4. Se \(x,y\in G\) então

    \[f(x^y)=f(y^{-1}\cdot x\cdot y)=f(y^{-1})\cdot f(x)\cdot f(y)=f(y)^{-1}\cdot f(x)\cdot f(y)=f(x)^{f(y)}.\]

Isto conclui esta argumentação.

É uma consequência da proposição acima que \(f(x^n)=f(x)^n\) para todo \(n\in\mathbb{Z}\).

Teoremas do Isomorfismo¶

Primeiro Teorema do Isomorfismo

Seja \(f:G\longrightarrow H\) um homomorfismo de gupos. Então:

  1. \(\ker_f=\{x\in G:f(x)=e_H\}\unlhd G\);

  2. \(\mathrm{Im}_f=\{y\in H:\exists\,\,x\in G,\,y=f(x)\}\leqslant H\);

  3. \(G/\ker_f\cong\mathrm{Im}_f\).

Prova

  1. Observe que:

    1. \(e_G\in\ker_f\) pois \(f(e_G)=e_H\);

    2. se \(x,y\in\ker_f\) então

      \[f(x\cdot y^{-1})=f(x)\cdot f(y^{-1})=e_H\cdot f(y)^{-1}=e_H^{-1}=e_H,\]

      e, assim, \(xy^{-1}\in\ker_f\);

    3. se \(x\in\ker_f,g\in G\) então

      \[f(x^g)=f(x)^{f(g)}=e_H^{f(g)}=e_H,\]

      de onde segue que \(x^g\in\ker_f\). Logo, \(\ker_f\unlhd G\);

  2. Observe que:

    1. \(e_H=f(e_G)\in\mathrm{Im}_f\);

    2. se \(x^\prime,y^\prime\in\mathrm{Im}_f\) então \(x^\prime=f(x),y^\prime=f(y)\) para certos \(x,y\in G\) e, assim, conclui-se que

      \[x^\prime\cdot(y^\prime)^{-1}=f(x)\cdot f(y)^{-1}=f(x)\cdot f(y^{-1})=f(x\cdot y^{\prime})\in\mathrm{Im}_f,\]

      pois \(x\cdot y^{-1}\in G\). Portanto, \(\mathrm{Im}_f\leqslant H\);

  3. Agora, considere a seguinte função:

    \[\bar{f}:G/\ker_f\longrightarrow\mathrm{Im}_f,\quad\ker_fx\longmapsto f(x).\]

    1. \(\bar{f}\) é de fato uma função:

      se \(x,y\in G\) e \(\ker_fx=\ker_fy\) então \(xy^{-1}\in\ker_f\), de onde segue que

      \[\bar{f}(\ker_fx)=f(x)=f(xy^{-1}\cdot y)=f(xy^{-1})\cdot f(y)=e_H\cdot f(y)=f(y)=\bar{f}(\ker_fy),\]

      e \(\bar{f}\) é, como se vê, uma bem definida função;

    2. \(\bar{f}\) é um homomorfismo de grupos:

      se \(x,y\in G\) então

      \[\bar{f}(\ker_fx\cdot\ker_fy)=\bar{f}(\ker_fx\cdot y)=f(x\cdot y)=f(x)\cdot f(y)=\bar{f}(\ker_fx)\cdot\bar{f}(\ker_fy);\]

    3. \(\bar{f}\) é injetora:

      se \(x\in G\) então

      \[\bar{f}(\ker_fx)=e_H\implies f(x)=e_H\implies x\in\ker_f\implies\ker_fx=\ker_f,\]

      de modo que \(\ker_{\bar{f}}=\{\ker_fx\in G/\ker_f:\bar{f}(\ker_fx)=e_H\}=\{\ker_f\}\) e como tal, \(\bar{f}\) é injetora. Com efeito, se \(x,y\in G\) então

      \[\begin{split}\begin{align} & \bar{f}(\ker_fx)=\bar{f}(\ker_fy)\implies\bar{f}( \ker_fx\cdot\ker_fy^{-1})=e_H\implies\ker_fx\cdot\ker_fy^{-1}\in \ker{\bar{f}}=\{\ker_f\}\implies \\ & \implies\ker_fx\cdot\ker_fy^{-1}=\ker_f\implies\ker_fx=\ker_f \cdot\ker_fy=\ker_fy; \end{align}\end{split}\]

    4. \(\bar{f}\) é sobrejetora:

      se \(y\in\mathrm{Im}_f\) então \(y=f(x)\) para algum \(x\in G\) e, assim,

      \[\bar{f}(\ker_fx)=f(x)=y,\]

      de onde segue que \(\bar{f}\) é uma função sobrejetora.

Segundo Teorema do Isomorfismo

Sejam G um grupo, \(H\leqslant G\) e \(N\unlhd G\). Então \(H\cap N\unlhd H\) e \(\frac{HN}{N}\cong\frac{H}{H\cap N}\).

Prova Exercício.

Terceiro Teorema do Isomorfismo

Sejam G um grupo, \(M\unlhd G\) e \(N\unlhd G\). Suponha que \(N\subseteq M\). Então \(M/N\unlhd G/N\) e \(\frac{G/N}{M/N}\cong\frac{G}{M}\).

Teorema da correspondência¶

Teorema

Seja \(f:G\longrightarrow H\) um epimorfismo de grupos. Se

\[\mathfrak{G}_f=\{M:M\leqslant G,\,M\supseteq\ker_f\}\quad\mbox{e}\quad\mathfrak{H}_f=\{W:W\leqslant H\},\]

então

\[\begin{split}\begin{array}{ccrclcccrcl} \Phi & : & \mathfrak{G}_f & \longrightarrow & \mathfrak{H}_f & \mbox{e}\quad & \Psi & : & \mathfrak{H}_f & \longrightarrow & \mathfrak{G}_f, \\ & & K & \longmapsto & f(K) & & & & W & \longmapsto & f^{-1}(W) \end{array}\end{split}\]

são funções inversas uma à outra. Mais ainda, se \(M\in\mathfrak{G}_f\) e \(W\in\mathfrak{H}_f\) então:

  1. \[\dfrac{M}{\ker_f}\cong\Phi(M)\quad\mbox{e}\quad\dfrac{\Psi(W)}{\ker_f}\cong W;\]
  2. \[M\unlhd G\implies\Phi(M)\unlhd H\quad\mbox{e}\quad W\unlhd H\implies\Psi(W)\unlhd G.\]

Prova Exercício.

Teorema de Cayley¶

Teorema

Sejam G um grupo e \(\mathrm{Sym}(G)\) o grupo de todas as bijeções \(f:G\longrightarrow G\) com respeito à operação de composição de funções. Então,

\[\Phi:G\longrightarrow\mathrm{Sym}(G),\quad g\longmapsto\Phi_g,\]

onde

\[\forall\,\,g\in G:\qquad\Phi_g:G\longrightarrow G,\quad x\longmapsto xg^{-1},\]

é um monomorfismo de grupos. Segue do Primeiro Teorema do Isomorfismo que:

\[G\cong\mathrm{Im}_\Phi\leqslant\mathrm{Sym}(G).\]

Prova

Vejamos que:

  1. \(\Phi_g:G\longrightarrow G\) é uma bijeção para cada \(g\in G\):

    1. se \(x,y\in G\) então

      \[\Phi_g(x)=\Phi_g(y)\iff xg^{-1}=yg^{-1}\iff x=y,\]

      de forma que \(\Phi_g\) é uma função injetora;

    2. se \(y\in G\) então \(x=yg\in G\) e

      \[\Phi_g(x)=xg^{-1}=(yg)g^{-1}=y,\]

      de maneira que \(\Phi_g\) é sobrejetora.

    Portanto, \(\Phi_g\in\mathrm{Sym}(G)\) para todo \(g\in G\);

  2. \(\Phi_{gh}=\Phi_g\circ\Phi_h\) para todos \(g,h\in G\):

    \[\forall\,\,x\in G:\quad\Phi_{gh}(x)=x(gh)^{-1}=(xh^{-1})g^{-1}=\Phi_g(\Phi_h(x))= \Phi_g\circ\Phi_h(x),\]

    de modo que \(\Phi_{gh}=\Phi_g\circ\Phi_h\) para todos \(g,h\in G\);

  3. \(\Phi_g\) é igual à função identidade \(\mathrm{id}_G:G\ni x\longmapsto x\in G\) apenas se \(g=e_G\in G\):

    se \(\Phi_g=\mathrm{id}_G\) então, dado \(x\in G\) qualquer, tem-se que:

    \[\Phi_g(x)=\mathrm{id}_G(x)\iff xg^{-1}=x\iff g^{-1}=e_G\iff g=e_G,\]

    de onde segue que \(g=e_G\).

As observações acima nos permitem dizer que \(\Phi:G\longrightarrow\mathrm{Sym}(G)\) é um monomorfismo de grupos, o que conclui esta argumentação.

Salientamos que para grupos finitos, digamos \(\vert G\vert=n\), tem-se que

\[G\cong\mathrm{Im}_\Phi\leqslant\mathrm{Sym}(G)\cong S_n,\]

pelo Teorema de Cayley acima (ou seja, G é isomorfo a um dos subgrupos do grupo simétrico n). A seguir apresentamos uma generalização do Teorema de Cayley.

Teorema

Sejam G um grupo e \(H\leqslant G\). Se \(\mathrm{Sym}(G/H)\) denota o grupo de todas as bijeções \(f:G/H\longrightarrow G/H\) com respeito à operação de composição de funções, então

\[\Phi:G\longrightarrow\mathrm{Sym}(G/H),\quad g\longmapsto\Phi_g,\]

onde

\[\forall\,\,g\in G:\quad\Phi_g:G/H\longrightarrow G/H,\quad Hx\longmapsto Hxg^{-1},\]

é um homomorfismo de grupos cujo kernel é justamente

\[Cor_G(H)=\bigcap_{x\in G}H^x,\]

o coração de H em G.

Prova Exercício.
